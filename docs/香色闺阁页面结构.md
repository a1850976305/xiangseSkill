## 📚 新建站点页面

### 🔧 站点基本信息

| 字段 | 说明 | 备注 |
|------|------|------|
| **站点名** | 站点名称，必须具有唯一性 | 用于标识不同的书源站点 |
| **host** | 站点首页地址，网址格式，必须唯一 | 所有 URL 将检测是否包含该 host，若未包含，则默认使用当前 host。可通过 `config.host` 获取 |
| **权重** | 权重值，用于排序或优先级控制 | 数值越高优先级越高 |
| **类型** | 站点内容类型 | 文本/小说、图片/漫画/壁纸、音乐/音频/听书、视频/电影/电视剧 |
| **备注** | 站点说明或备注信息 | 可选填写 |
| **httpHeaders** | 全局 HTTP 请求头（JSON 格式） | 可通过 `config.httpHeaders` 获取 |

### 📖 书源配置区域

#### 🔍 书籍搜索配置

**📋 请求信息配置**
> 书籍详情/章节列表/章节内容/书单详情等子级配置一般不用填写，会自动从上一级获取url。

**🔧 配置方法**

**方法1：使用替换规则**
```
可替换的参数：
- %@result (从上一级获取的地址)
- %@keyWord (搜索关键词)
- %@pageIndex (页码)
- %@offset (偏移量)
- %@filter (过滤条件)

使用示例：
https://www.host.com/search?key=%@keyWord&p=%@pageIndex
```

**方法2：使用JS动态配置**
```javascript
// 需使用@js:声明，默认js参数有config/params/result
@js:
let hp = {};
hp.key = params.keyWord;
hp.p = params.pageIndex;
let url = 'https://www.host.com/search';
let hh = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/535.1 (KHTML, like Gecko) Chrome/14.0.835.163 Safari/535.1',
    'Referer': 'https://www.baidu.com'
};
return {'url': url, 'httpParams': hp, 'httpHeaders': hh};
```

**🔑 常用配置参数**

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `url` | 字符串 | 要请求的url | 必填 |
| `POST` | 布尔值 | 是否使用POST请求 | false (使用GET) |
| `httpParams` | 字典 | HTTP参数 | {} |
| `httpHeaders` | 字典 | HTTP请求头 | {} |
| `forbidCookie` | 布尔值 | 是否禁用cookie | false |
| `webView` | 字符串 | 使用webView请求完整网页 | 空 |
| `webViewJs` | 字符串 | webView请求完成后执行的js | 空 |
| `webViewJsDelay` | 整数 | webView完成后延时执行js（秒） | 1 |
| `webViewSkipUrls` | 数组 | webView要跳过的url | [] |
| `sourceRegex` | 字符串 | 嗅探资源url的正则表达式 | 空 |
